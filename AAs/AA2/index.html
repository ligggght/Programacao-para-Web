SOBRE O PROJETO

O Master Mind é um jogo clássico de lógica e dedução onde o jogador deve descobrir uma sequência secreta de cores, criada por um segundo jogador. Este projeto implementa uma versão digital do jogo utilizando tecnologias web modernas com estética retrô 8-bit.

PEQUENO PREVIEW DO JOGO

OBJETIVOS

  • Implementar uma versão digital do jogo Master Mind
  • Aplicar conceitos de Programação para Web com HTML5, CSS3 e JavaScript
  • Utilizar o framework React para criar uma interface interativa
  • Desenvolver habilidades em design responsivo e UX
  • Implementar lógica de jogo e algoritmos de verificação
  • Criar uma estética retrô 8-bit autêntica

COMO JOGAR

1. OBJETIVO

Descubra a sequência secreta de 4 cores em até 10 tentativas. Use sua lógica e dedução para quebrar o código!

2. CORES DISPONÍVEIS

6 cores neon disponíveis para criar suas combinações.

3. SISTEMA DE FEEDBACK

cor correta na posição correta

cor correta na posição errada

use o feedback para ajustar sua próxima tentativa!
